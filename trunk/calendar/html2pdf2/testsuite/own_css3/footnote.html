Abstracttaken from CSS 3 draft at w3c.org

This module describes features often used in printed publications. In particular, this specification describes how CSS style sheets can express named strings, leaders, cross-references, footnotes, endnotes, running headers and footers, named flows, ad hoc counter styles, paged-based floats, hyphenation, change bars, named page lists, and generated lists. Along with two other CSS3 modules — multicolumn layout and paged media — this module offer a way of presenting structured documents on paged media.these specs are incomplete!
