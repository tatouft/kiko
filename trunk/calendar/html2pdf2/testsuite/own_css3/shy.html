This module describes the page model that partitions a flow into pages. It builds on the CSS3 Box model mo­du­le and introduces and defines the page model and paged media. It adds functionality for pagi­nation, page mar­gins, headers and footers, image ori­en­ta­tion. Fi­nal­ly it extends ge­ne­ra­ted content for the purpose of cross-re­fe­ren­ces with page numbers.
